Common Specifications

The specs reported below are common to both transmitter versions:

  • Dimension: 200*194*110mm (L*W*H)
  • Weight: 670g (without battery)
  • Operating system: OpenTX
  • Mode: Mode 2
  • Number of channels: 24 channels
  • Internal RF module: ISRM-S-X9
  • Operating voltage range: 6.5 - 8.4V
  • Operating current: 130mA@8.2V
  • Operating Temperature: -20℃ ~ 60℃ (-4℉ ~ 140℉)
  • Backlit LCD resolution: 212*64
  • Smart Port, Micro SD card slot and DSC Port
  • Micro USB interface: supports 2S Li-battery balancing charge
  • Model memories: 60 models (expandable by Micro SD card)
  • Compatibility: ACCST D16 and ACCESS receivers (ACCST D8 is not available, an external module is required)

Main upgrades in 2019 version

The following image show the FrSky Taranis X9D Plus SE 2019:

FrSky Taranis X9D Plus 2019 (Standard or SE) is the upgraded version of the notorious Taranis X9D Plus. The most important changes are the following:

  • use of ACCESS (Advanced Communication Control, Elevated Spread Spectrum) RC control and communications protocol;
  • power on/off is achieved by a button rather than a slider;
  • additional momentary button placed on the top left useful for DLG pilots to activate launch mode;
  • program scroll wheel on the right of the screen substituting buttons to make it easier to navigate the menus;
  • upgraded MCU is applied to a re-structured mainboard that further increase the compute capability and extend the storage resource of data: this is to allow the execution of more complex LUA scripts and it also optimize the overall performance (for example, for voice speech outputs).

Everything runs, like with the previous model, on the solid OpenTX firmware.

ACCESS protocol

The ACCESS communication protocol boasts 24 channels with a faster baud rate and lower latency with a high-speed module digital interface. Along with the new spectrum analysis function added to the OpenTX firmware, it is now possible to check the airwaves for RF noise with an accurate SWR indicator, which can measure yours and others signal strength in the area. So, the Spectrum Analyzer allows the user to measure power, noise, and distortion of the frequencies around.

Moreover, thanks to the ACCESS protocol, once a receiver is registered to a radio, there is no need to access the receiver to bind (pressing bind button) or flash a new firmware (via smartport) since these procedures are achieved through wireless technology (Smart Match and Smart Share features).

FrSky also states that the protocol has a better security due to its higher encryption, assuring model’s signal will not be interfered with.

Internal RF Modes

These are the internal RF modes supported in Taranis X9D Plus 2019:

  • ACCESS;
  • ACCST D16;
  • ACCST LR12.

External RF Modes

Taranis X9D Plus 2019 supports the addition of external modules, like FrSky R9M or TBS Crossfire. OpenTX firmware installed by default (version 2.3.0) does not have the option for TBS; available external RF modes are:

  • PPM;
  • XJT D16;
  • DSM2 LP45;
  • R9M FCC;
  • SBUS.

So, to use TBS Crossfire, flashing the latest firmware is required (CSRF will appear in modes list).

Battery

Taranis X9D Plus 2019 comes with no battery included. Since its working voltage is 6.5 - 8.4V, it can be powered with 2S LiPo batteries, but no 3S (differently from the previous Taranis X9D).
Here are reported three batteries which are compatible with the specs and the space of the Taranis X9D Plus 2019 battery compartment:

  • 2S 7.4V 3000mAh LiPo battery
  • ZOP Power 7.4V 3000mah 10C Lipo Battery
  • Gaoneng GNB 7.4V 2500mAh 5C 2S Lipo Battery

They assure a pretty decent working time.

Gimbal Stick Ends

FrSky Taranis X9D Plus SE 2019 gimbal stick ends are made with very good material, but they are more suitable for thumbs pilots; if you are a pinch pilot instead, you can change them with M4 aluminum transmitter classic stick ends.

Conclusions

FrSky Taranis X9D Plus 2019 is a very good transmitter, especially considering the promising new ACCESS protocol, that could simplify many tasks. OpenTX firmware is not totally ready yet, but it is just a matter of time.
